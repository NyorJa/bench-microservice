# bench-microservice

## Spring microservices using spring cloud (config, eureka, gateway, circuit breaker, zipkin-sleuth)

### Prerequisites
- Java 11
- Maven 3++
- Docker
- Vault
- Keycloak
- Zipkin
- OATH2

### How to run
- Run configuration-server, discovery-server, order-service, product-service, gateway-service, notification service
